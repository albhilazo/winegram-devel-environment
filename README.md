# Winegram development environment

Vagrant + Puppet environment for the Winegram project.

Based on an `ubuntu/trusty64` (Ubuntu 14.04 LTS) vagrant box.

Includes:

 - Apache `2.4.20`
 - PHP `5.6.20` (with composer)
 - Mysql `5.5.49`
 - Redis `2.8.4`
